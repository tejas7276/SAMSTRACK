# 📚 SAMSTRACK | Student Attendance Management System

> Automating student attendance with Spring Boot, Hibernate, MySQL, and Angular

---

## 🛠️ Tech Stack
- **Backend:** Java, Spring Boot, Hibernate
- **Frontend:** Angular
- **Database:** MySQL
- **Architecture:** RESTful APIs

---

## 📖 About the Project

**SAMSTRACK** is a full-stack Student Attendance Management System designed to automate and simplify attendance tracking in educational institutions.

It provides an intuitive platform for faculty to:
- Record attendance
- Manage students and subjects
- Track attendance data securely and accurately

Built with a modern tech stack to ensure scalability, reliability, and ease of integration with other educational platforms.

---

## 🚀 Features

- 🎯 Faculty can record and manage attendance by subject.
- 📚 Manage student and subject information efficiently.
- 🧾 Store attendance records securely with automated validation.
- ✅ Real-time data accuracy and consistency.
- 🔗 Easy integration with existing educational systems.

---

## 🗂️ Core Entities

- **Student**: Information about enrolled students.
- **Subject**: Subjects offered in the institution.
- **Faculty**: Teachers who manage subjects and attendance.
- **Attendance Record**: Daily tracking of student attendance.
